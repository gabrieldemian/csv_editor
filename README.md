# BootlegEditor3000

Test task of EOTLabs.

## How to use
Run the project with `cargo run`. A TUI will render, and you can control it using the following keys:

`h` move left
`l` move right
`j` move down
`k` move up

`e` edit cell
`d` delete cell
`q` quit

On update and delete, the csv file will be updated.
